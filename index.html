    How to Edit Marzenapedia

    Writing Articles

    Articles live in articles/ as .md files. Click the Edit tab on any article to open the editor. The editor supports a split Write | Preview mode — switch tabs at the top of the editor.

    When you click Save to GitHub, the page opens GitHub and copies the content to your clipboard. Paste and commit — the index rebuilds automatically within ~30 seconds.

    Marzena Commons

    All images are managed in the Commons (nav bar above). Upload via the GitHub uploader, optionally add metadata in images/commons.json. Drag images from the Commons picker directly into the editor to insert them.
