    How to Edit Marzenapedia

    Writing Articles

    Articles live in articles/ as .md files. Click the Edit tab on any article to open the editor. When you click Save to GitHub:

    • New articles — GitHub opens with the filename and content pre-filled. Just commit.
    • Existing articles — GitHub opens the file editor. The content is copied to your clipboard; select all and paste.

    When you commit, the GitHub Action rebuilds index.json automatically so your changes appear in search and on the homepage within ~30 seconds.

    Image Paths

    Images in images/ can be referenced as bare filenames: CoAMZ.png or FlagMZ.jpg. Or use the full path: /images/CoAMZ.png.
