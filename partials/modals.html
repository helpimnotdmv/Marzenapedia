    Upload an Image to Commons

    Images live in the images/ folder of the repository. To add a new image:

    1. Click the link below to open the GitHub uploader.
    2. Drag your image file onto the page, or click "choose your files".
    3. Commit the file — it will appear in the Commons within ~30 seconds.
    Upload to images/ on GitHub →
    Add title & description (optional but recommended):
    Edit or create images/commons-meta.json in the repo. Each entry looks like:
    { "filename": "FlagMZ.jpg", "title": "Flag of Marzena", "description": "The national tricolour, adopted 1952." }
    The build process merges this into commons.json automatically.
    How to Edit Marzenapedia

    Writing Articles

    Articles live in articles/ as .md files. Click the Edit tab on any article to open the editor. The editor supports a split Write | Preview mode.

    When you click Save to GitHub, the page opens GitHub and copies the content to your clipboard. Paste and commit — the index rebuilds automatically within ~30 seconds.

    Marzena Commons

    All images are managed in the Commons (nav bar above). Upload via the GitHub uploader, optionally add metadata in images/commons-meta.json. Drag images from the Commons picker directly into the editor to insert them.
