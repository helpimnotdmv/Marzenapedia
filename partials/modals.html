    Upload an Image to Commons

    Images live in the images/ folder of the repository. To add a new image:

    1. Click the link below to open the GitHub uploader.
    2. Drag your image file onto the page, or click "choose your files".
    3. Commit the file — it will appear in the Commons within ~30 seconds.
    Upload to images/ on GitHub →
    Add title, description, and tags (recommended):
    Edit images/commons-meta.json in the repo. Each entry looks like:
    {
  "filename": "FlagMZ.jpg",
  "title": "Flag of Marzena",
  "description": "The national tricolour, adopted 1952.",
  "credit": "National Archive",
  "tags": ["flag","national symbol"]
}
    Image dimensions and file size are detected automatically by the build script.

    • Click Tools in the top bar to find the Editor, Graph Tool, and Stats Tool.

    • Edit any article via its Edit tab.
